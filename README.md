## ⚠️ 錯誤回報 (Bug Report)

如果你在使用過程中遇到任何問題，請前往 [Issues](https://github.com/K-en-u/RespawnDisconnect/issues) 頁面提交回報。

回報時請提供以下資訊：

### 1. 基礎版本資訊 (必要)
例如
* **Mod 版本**：`RespawnDisconnect-1.21.11-1.0`
* **遊戲版本**：`1.21.1` (或你使用的對應版本)
* **Fabric Loader 版本**：(例如 0.16.9)

### 2. 錯誤日誌與報告(非必要，也可以不提供)
如果遊戲崩潰 (Crash) 或出現錯誤訊息，請提供以下檔案：
* **錯誤報告 (Crash Report)**：通常位於 `.minecraft/crash-reports/` 資料夾。
* **遊戲日誌 (latest.log)**：位於 `.minecraft/logs/latest.log`。
