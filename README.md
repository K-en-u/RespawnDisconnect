## ⚠️ 錯誤回報 / Error Reporting /エラー報告
<details>
  <summary><b>繁體中文</b></summary>
  <br>
## ⚠️ 錯誤回報

如果您在使用過程中遇到任何問題，請前往 [Issues](https://github.com/K-en-u/RespawnDisconnect/issues) 頁面提交回報。
回報時請提供以下資訊：
* **Mod 版本** (必要)：例如 `RespawnDisconnect-1.21.11-1.0`
* **遊戲版本** (必要)：例如 `1.21.11`
* **錯誤/異常內容**：請敘述發生的問題，或直接貼上遊戲內的錯誤訊息。
* **錯誤日誌/報告** (非必要)：`crash-reports` 或 `latest.log`。
</details>

<details>
  <summary><b>English</b></summary>
  <br>
## ⚠️ Error Reporting

If you encounter any issues, please report them on the [Issues](https://github.com/K-en-u/RespawnDisconnect/issues) page.
Please provide the following information:
* **Mod Version** (Required): e.g., `RespawnDisconnect-1.21.11-1.0`
* **Game Version** (Required): e.g., `1.21.11`
* **Error/Exception Details**: Please describe the issue or paste the error message.
* **Error Logs/Reports** (Optional): `crash-reports` or `latest.log`.
</details>

## ⚠️ エラー報告
  
問題が発生した場合は、[Issues](https://github.com/K-en-u/RespawnDisconnect/issues) ページで報告してください。
報告の際は、以下の情報を提供してください：
* **Mod バージョン** (必須): 例 `RespawnDisconnect-1.21.11-1.0`
* **ゲームバージョン** (必須): 例 `1.21.11`
* **エラー/異常内容**: 発生した問題の説明、またはエラーメッセージを貼り付けてください。
* **エラーログ/レポート** (任意): `crash-reports` または `latest.log`。
</details>
